Fake Church Website (Embed Test Harness)

Simulates a third-party site with hostile CSS, to verify the embed loader is resilient to it.

1. Donate Button (opens secure popup)

2. Second Button (multiple buttons on one page)

3. Script Loaded Twice (dedupe guard)

4. Inline Embed (renders a Give button directly in the page)

5. Invalid Slug (popup shows a friendly error)
